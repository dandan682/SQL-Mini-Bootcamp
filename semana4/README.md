# NIEVA SQL-Mini-Bootcamp.

__En este ejercicio de semana 4 es practicar los siguientes conceptos.__

* JOINs como tema: INNER, LEFT, RIGHT, SELF, CROSS.
* Combinar 3, 4, 5 tablas con confianza.
* Detectar huérfanos con LEFT JOIN ... WHERE x IS NULL.
* Alias y buenas prácticas de queries multi-tabla.
