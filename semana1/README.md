# NIEVA SQL-Mini-Bootcamp.

__En este ejercicio de semana 1 es practicar las siguientes acciones de SQL.__


* Eliminar base de datos anterior (si existe): DROP DATABASE IF EXISTS nombre_base;
* Crear una base de datos: CREATE DATABASE nombre_base_datos;
* Seleccionar una base de datos: USE nombre_base_datos;
* Crear una tabla: CREATE TABLE nombre_tabla (columna1 TIPO_DATO,columna2 TIPO_DATO,...);
* Insertar datos en una tabla: INSERT INTO nombre_tabla (columna1, columna2, ...) VALUES (valor1, valor2, ...);
* Consultar datos de la tabla: SELECT * FROM nombre_tabla;
* Ver columnas específicas: SELECT columna1, columna2 FROM nombre_tabla;
* Filtrar filas que cumplen condiciones: WHERE condiciones [IS NULL | IS NOT NULL]
* Verificar si un valor está en una lista: IN (valor1, valor2, valor3);
* Buscar texto que coincida con un patrón: LIKE '%__%';
* Ordenar resultados: ORDER BY columna [ASC | DESC];
* Limitar resultados: LIMIT número;
* Valores únicos: DISTINCT
